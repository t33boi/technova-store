# TechNova — Premium IT Gadgets Store 🇳🇬

A fully functional, multi-page ecommerce website for selling IT gadgets like phones, laptops, cameras, audio equipment, and wearables. Built for the Nigerian market with Naira (₦) pricing and local context.

![TechNova Preview](https://images.unsplash.com/photo-1468495244123-6c6c332eeece?w=800&q=80)

## 🌐 Live Demo

[View on Vercel →](https://technova-store.vercel.app/)

## ✨ Features

- **Multi-page** — Home, Shop, Product Detail, Cart, Deals, Blog, About, Contact, FAQ
- **Dynamic Product Catalog** — 20+ products with filtering, sorting, search & pagination
- **Shopping Cart** — Add/remove items, quantity controls, coupon codes (TECH10, NOVA20, SAVE5K)
- **Product Detail** — Image zoom, tabs (description/specs/reviews), related products
- **Hero Carousel** — Auto-sliding banner with animated transitions
- **Deals Page** — Countdown timers, featured deals, promo banners
- **Blog** — Expandable articles with search and category filtering
- **Contact Form** — With validation and interactive Google Maps placeholder
- **FAQ** — Accordion-style with search functionality
- **Scroll Animations** — Elements animate into view on scroll
- **Fully Responsive** — Mobile, tablet, and desktop layouts
- **Dark Navy Theme** — Modern UI with gradient accents

## 🛠️ Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** — No frameworks, no dependencies
- **Font Awesome 6** — Icons
- **Google Fonts** — Outfit + Space Mono
- **Unsplash** — Product and lifestyle images

```
├── faq.html          # FAQ accordion with search

```

## 🚀 Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/YOUR_USERNAME/technova-store.git
   cd technova-store
   ```

2. **Open locally**
   Just open `index.html` in your browser, or use a local server:
   ```bash
   npx serve .
   ```

3. **Deploy to Vercel**
   - Push to GitHub
   - Import repo at [vercel.com/new](https://vercel.com/new)
   - Deploy — done!
